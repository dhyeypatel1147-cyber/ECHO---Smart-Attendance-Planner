🚀 ECHO - Smart Attendance Planner

A modern, responsive web app that helps students **track attendance smartly** and avoid shortage.

---

📌 Features

* 📊 Calculate attendance percentage instantly
* 🎯 Know how many classes to **attend or skip**
* 😎 Smart status system (Safe / Warning / Critical)
* 🎨 Clean futuristic UI with glassmorphism
* 📱 Fully responsive (mobile + desktop)
* 🧠 Dynamic mood-based visuals

---

🛠️ Tech Stack

* HTML
* CSS
* JavaScript

⚙️ How It Works

1. Enter total classes
2. Enter attended classes
3. Enter required attendance (%)
4. Click **Calculate**

👉 The app will:

* Show your current attendance
* Tell you how many classes to attend or skip
* Display your status
